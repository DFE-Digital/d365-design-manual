Create new web file records

Before continuing, make sure you have downloaded the portal files.

For each of the downloaded web files you need to create a new web file record, with the exception of theme.css (go to updating your theme file).

  1. Navigate to the Portal management model-driven app in your D365 environment
  2. Select Web files from the left-hand page navigation
  3. Create a new web file record for each of the downloaded files filling out the following properties:

    Name: Enter the file name as it was downloaded, including extension e.g. dfe-crm-portal-helper.js
    Website: Select your portal website record
    Parent Page: Select your home page
    Partial Url: Enter the file name as it was downloaded, including extension e.g. dfe-crm-portal-helper.js
    Publishing state: Select Published

  4. In the Notes tab, attached the respective downloaded file as a note file attachment

Update your theme file

Follow steps 1 and 2 above, then find the web file record called theme.css and open it.

In the Notes tab, replace the attached file with your downloaded theme.css file and save the record
