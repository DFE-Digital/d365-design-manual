To create a site (portal) in your environment, sign in to Power Apps and follow these steps.

  1. Select the correct environment in the top toolbar (top right).
  2. On the left pane, select Create.
  3. Select Blank app at the top of the page.
  4. Click Create under 'Power Pages Website'.
  5. In the create portal window, enter a name for the portal and address for the website, and select a language from the drop-down list. If you've selected to create portal from blank, you can optionally choose a website record.
  6. Select Create.

Once your portal has been created, download the base portal files then proceed to set up your portal.
