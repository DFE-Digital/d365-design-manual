How it works

A panel card and be positioned anywhere on a page.

The panel card by default will be the same width as the page's other content. For example, if the other content takes up two-thirds of the screen on desktop devices, then the panel card component will also take up two-thirds. {{ panelCardExample() }}

Including hyperlinks

If you are including a hyperlink in your panel card component use the dfe-panel__link class within your <a> tag.

Grid layout

If you wish to make the panel cards smaller, confine the component within a grid system. Read about grid systems (opens in a new tab).

The example below shows 3 panel card components in a 3 column grid row. By default, the panel card for each column will match the height of the content.

{{ panelCardGridExample() }}

Panel card flex height

There may be occasions where you need to make all the panel cards in a grid system row the same height.

you can add additional classes to the component so all the panel cards in the same row equal the height of the card with the most content.
