When to use this component

Use the card component to chunk up related content, data and links. They can also be used to highlight content on a page.

How it works

A panel card and be positioned anywhere on a page.

The panel card by default will be the same width as the page's other content. For example, if the other content takes up two-thirds of the screen on desktop devices, then the panel card component will also take up two-thirds. If you wish to make the panel cards smaller, confine the component within a grid system. Read about grid systems (opens in a new tab)..

Including hyperlinks

If you are including a hyperlink in your panel card component use the class dfe-panel__link within your <a> tag.
